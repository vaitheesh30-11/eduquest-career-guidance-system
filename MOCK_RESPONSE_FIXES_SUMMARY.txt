HOW IT WORKS:
================================================================================

When Gemini API is unavailable:
1. System detects API unavailability gracefully
2. BaseLLMAgent returns intelligent fallback based on prompt context
3. Each agent (alternatives, market, reality) provides career-specific content
4. All responses are professional, helpful, and career-focused
5. No generic "Mock LLM" or "Fallback" messages appear to users

Example Flow:
Input: Dream Career = "Data Scientist"
Viability Score: 0.72, Academic Fit: 73%

Output:
- Market Insights: Data Science specific demand/salary/growth
- Reality Check: Assessment scaled to 72% viability
- Alternatives: ML Engineer, Data Analyst, Analytics Engineer (all realistic)
- All content is career-specific and actionable
